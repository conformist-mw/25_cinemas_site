На сайте доступно публичное api в json формате.

Для получения данных о всех фильмах перейдите по адресу весь список.

Для получения данных по одному фильму введите в адресной строке http://domain.tld/api/movie/number, где number — обычно шестизначное число.

Пример:

$ curl -I http://localhost:5000/api/movie/226062/
HTTP/1.0 200 OK
Content-Type: application/json; charset=utf-8
Content-Length: 6506
Server: Werkzeug/0.11.11-dev Python/3.5.2+
Date: Sun, 08 Jan 2017 10:11:33 GMT
Настоящий сайт создан в исключительно учебных целях. Все сведения о фильмах взяты с сайта Афиша.
